# ALconnects — Personal Portfolio 🚀

> A sleek, dark-themed developer portfolio website for **Arya Lolusare** — AI/ML Engineer from Nagpur, India. Built entirely with Python and vanilla HTML/CSS, zero frameworks, zero dependencies.

---

## ✨ Features

- **Single-file Python server** — runs instantly with no setup
- **Custom animated cursor** with smooth ring-follow effect
- **Glowing orbital orb** with spinning tech rings on the hero section
- **Space Grotesk** typography — modern, geometric, professional
- **Scroll-reveal animations** on education timeline
- **Active nav highlighting** on scroll
- **Fully responsive** — mobile-friendly layout
- **Noise texture overlay** for depth and character
- Sections: Hero · About · Projects · Education · Contact

---

## 🛠️ Tech Stack

| Layer | Tech |
|-------|------|
| Server | Python `http.server` + `socketserver` |
| Frontend | Vanilla HTML5, CSS3 |
| Fonts | Space Grotesk, Space Mono, DM Serif Display |
| Animations | CSS keyframes + JS IntersectionObserver |

---

## 🚀 Getting Started

**Requirements:** Python 3.x (no pip installs needed)

```bash
# Clone the repo
git clone https://github.com/AryaLolusare2712/ALconnects.git
cd ALconnects

# Run the portfolio
python ALconnects.py
```

The browser opens automatically at **http://localhost:8080**

To stop the server: `Ctrl + C`
